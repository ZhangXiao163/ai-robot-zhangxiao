## Trot 步态原理

Trot 是四足机器人常见的小跑步态，其特点是对角腿成组运动：

| 相位 | 抬起/摆动腿 | 支撑腿 |
| :--- | :--- | :--- |
| Phase A | 左前腿、右后腿 | 右前腿、左后腿 |
| Phase B | 右前腿、左后腿 | 左前腿、右后腿 |

这种步态可以在速度和稳定性之间取得较好的平衡。

## 控制思路

1. 设置仿真环境、地面和机器人模型。
2. 根据时间生成周期性步态相位。
3. 为每条腿计算目标位置或关节角。
4. 将目标关节角发送给 PyBullet 控制接口。
5. 持续刷新仿真，观察机器人运动状态。
